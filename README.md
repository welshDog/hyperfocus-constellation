## 🧠 **What Makes This Special**

Traditional productivity tools fail neurodivergent minds. **Hyperfocus Zone** proves a different approach works better.

This isn't just a portfolio—it's a **living constellation** of 24 interconnected repositories, each designed for how ADHD and neurodivergent brains actually work. Every star tells a story of innovation born from hyperfocus.

**🎯 The Mission:** Prove that neurodivergent minds don't need fixing—they need better tools built for their cognitive superpowers.

## 🚀 **Quick Start - 3 Steps to Galaxy**

### 1. **Clone the Universe**
git clone https://github.com/welshDog/hyperfocus-constellation.git
cd hyperfocus-constellation


### 2. **Launch into Space**
Just open in your browser - no build process needed!
open index.html

Or use a local server:
python -m http.server 8000


### 3. **Explore the Galaxy**
- **Click stars** to visit repositories
- **Search** for specific projects  
- **Try Portal Mode** for the full experience
- **Use + and - keys** to zoom around

## 🎯 **How to Navigate Your New Galaxy**

### **Mouse Exploration**
- **Click stars** → Opens GitHub repository
- **Hover stars** → Shows project details
- **Drag zoom slider** → Scales the galaxy
- **Click Portal Mode** → Activates rotation

### **Keyboard Mastery** ⌨️
- **Tab** → Navigate between stars
- **Enter/Space** → Activate selected star
- **+/=** → Zoom in
- **-** → Zoom out  
- **Escape** → Close tooltips
- **/** → Focus search box

### **Screen Reader Support** 📢
- Full ARIA labeling for all interactive elements
- Live regions announce filter and search changes
- Logical navigation order
- Clear descriptions of all functionality
